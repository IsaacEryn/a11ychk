# Changelog — A11y Check 확장

형식: [Keep a Changelog](https://keepachangelog.com/ko/1.1.0/) · 버전: SemVer

## [0.4.0] - 2026-07-25

### 변경 (신규 기능)
- **수동 점검 체크리스트를 WCAG 2.2 성공기준 축으로 전환** — 항목마다 A/AA 수준
  배지와 KWCAG 대응 표기를 함께 표시. 검사 방법 안내는 대응 KWCAG 항목의 절차를
  그대로 따르며, WCAG 대응이 없는 KWCAG 고유 항목(5.4.3·6.4.4)은 "KWCAG 추가
  항목" 그룹으로 유지
- **판정이 사이트 통합 준수율에 반영** — 계정 저장 시 판정을 WCAG 성공기준으로
  기록해 보고서의 수동·통합 점수에 바로 반영됨 (기존에는 표시만 되고 점수에
  미반영이던 문제 해소)
- 기존 로컬 판정(KWCAG 항목 키)은 로드 시 대응 성공기준으로 자동 이전(멱등) —
  판정·메모 유실 없음

## [0.3.1] - 2026-07-22

### 변경 (자체 개선)
- **하위 프레임(iframe) 커스텀 검사**: 자동 검사(axe)처럼 자체 규칙 신호 수집도
  접근 가능한 하위 프레임까지 확대 — 프레임 안의 자동재생·작은 타깃·대체텍스트
  문제를 놓치지 않음 (건너뛰기 링크 등 페이지 구조 판정은 상위 문서 기준 유지)
- 페이지 리로드·클라이언트 라우팅 완료 시 패널의 대상 URL·체크리스트·결과 복원이
  더 안정적으로 따라가도록 갱신 트리거 보강

### 내부 구조 (동작 불변)
- 신호 수집기 정본을 core 패키지로 이관 — 서버 스캐너와 확장이 한 곳의 구현을
  공유하고 골든 테스트가 동일성을 강제
- 사이드패널 코드를 책임별 모듈(panel/state·session·scan·render·save·review·tools)로
  분해 (1,416줄 → 부트스트랩 109줄)
- 페이지 오버레이 공용 헬퍼(컨테이너·스크롤 추종)를 1회 설치 방식으로 통일 —
  4중 중복 코드 제거

## [0.3.0] - 2026-07-22

### 추가
- **가이드형 수동 판정**: 수동 점검 각 항목에 "가이드로 판정" — 확인 절차를 단계별로
  보여주고 예/아니오/판단 불가 답변으로 판정을 바로 기입 (자유 판정·메모와 공존)
- **초점 순서 판정 연동**: 도구 탭에서 초점 순서 오버레이를 켜면 판정 카드가 나타나
  확인 결과를 KWCAG 6.1.2 판정으로 즉시 저장 — 검사 탭 체크리스트에 바로 반영
- **확인 필요(needs review) 심사 흐름**: 자동 판정 불가 항목을 위반 목록 아래에 별도
  섹션으로 목록화 — 요소 위치 표시로 확인한 뒤 [위반 확정/문제없음] 결정, 점수 즉시 재계산
- **위반 목록 필터**: 심각도 칩(개수 표기)·규칙별 셀렉트로 위반 상세를 좁혀 보기
- **검사 결과 유지**: 탭을 옮기거나 패널을 닫았다 열어도 페이지별 최근 결과를 복원
  (검사 시각 안내와 함께 — 브라우저 종료 시 자동 삭제)
- **수동 점검 진행률**: 판정 완료 n/31 표시 + 진행 바 + 미판정만 보기 필터,
  4개 원칙(인식·운용·이해·견고)별 그룹 표시

### 변경 (자체 개선)
- 스크린리더 고지 개선: 검사 완료 요약·요소 표시 결과·판정 저장을 단일 상태 리전으로
  안내하고, 결과 영역 전체가 통째로 낭독되던 문제 해소
- 언어 설정이 판정 버튼·강조 라벨에 적용되지 않던 초기화 순서 문제 수정
- 페이지에 넣는 안내 문구 렌더 방식을 정리해 안전성 일관화
- 확장이 노출하는 리소스 선언(web_accessible_resources)을 제거 — 동작 변화 없이
  외부에서 확장 설치 여부를 탐지할 수 있는 표면 축소

## [0.2.1] - 2026-07-22 (웹스토어 첫 제출 후보)

### 추가
- 환경설정에 **언어 설정**(자동(시스템)/한국어/English) — 선택 즉시 패널 전체에 적용,
  axe 진단 언어·웹 링크 로케일도 함께 전환

### 변경
- 확장 이름에 '도구' 명시: "A11y Check — 웹 접근성 점검 도구" (en: Audit Tool)
- 비로그인 무료 검사 한도: 일 3회 → **주 3회** (월요일 시작, 로컬 집계)
- 검사 진행 표시가 완료 후에도 남는 문제 수정([hidden] CSS 보강)
- 도구 칩 활성화 시 ✓ 추가로 폭이 변해 옆 칩이 밀리던 문제 수정(체크 공간 상시 확보) · 도구 그룹 상단 여백 추가
- 위반 표시 마커·개별 강조가 선택자 첫 매치만 사용하던 문제 수정 — 매치 전부에 표시(중복 구조 페이지에서 엉뚱한 요소 표시 방지)
- 로그인 확장 검사 한도가 등급별로 적용(무료 10·프로 20·엔터 30/일) — 가입 유도 문구도 일치
