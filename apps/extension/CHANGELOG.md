# Changelog — A11y Check 확장

## [0.2.0] - 2026-07-21 (웹스토어 첫 제출 후보)

### 추가
- Side Panel 셸(검사/도구/환경설정 탭) + 테마 시스템
- 시각 도구: 위반 표시 오버레이(스크롤 추종)·시각 시뮬레이션·구조 보기·명도 대비 스포이드
- 타깃 크기 점검, 수동 항목별 맞춤 시각 강조 토글
- 패널 내 전문가 판정 + 웹 계정 저장(사이트 보고서 병합)
- AI 수정 요청 내보내기(MD/JSON), axe 진단 한국어화
- ko/en 다국어(_locales)

### 변경
- 사용량 체계: 익명(로컬 3회 넛지)/회원(서버 집계) 분리
- 브랜드 표기 "A11y Check" 통일
